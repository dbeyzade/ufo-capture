# UFO Capture - Subscription Features

## Overview
UFO Capture offers a trial-and-unlock model via in-app purchase, allowing users to experience core functionality before committing to the full feature set.

## Subscription Tiers

### Free Trial (Default)
**Duration:** 7 days
**Features:**
- ✓ Basic motion detection
- ✓ Standard RGB camera mode
- ✓ Manual recording (up to 5 minutes per session)
- ✓ Local gallery (last 5 recordings)
- ✓ Basic notifications
- ✗ Advanced color modes (Infrared, UV, Night Vision)
- ✗ External stream support
- ✗ Unlimited recording time
- ✗ Advanced motion sensitivity settings

### Premium Subscription
**Price:** $9.99/month or $79.99/year
**Features:**
- ✓ All Free Trial features
- ✓ **Advanced Color Modes:**
  - Infrared mode (red-channel emphasis)
  - Ultra Violet mode (violet emphasis)
  - Night Vision (low-light enhancement)
  - CMYK mode (color separation)
  - Black & White (contrast enhancement)
- ✓ **Unlimited Recording:**
  - No time limits per session
  - Unlimited storage (device-dependent)
  - Background recording support
- ✓ **External Stream Support:**
  - HTTP/HTTPS IP camera integration
  - Wi-Fi camera compatibility
  - Multi-stream monitoring (coming soon)
- ✓ **Advanced Motion Detection:**
  - Customizable sensitivity levels
  - Zone-based detection
  - Smart object tracking
- ✓ **Enhanced Gallery:**
  - Unlimited recording history
  - Cloud backup option (coming soon)
  - Advanced filtering and search
- ✓ **Priority Support:**
  - Email support within 24 hours
  - Early access to new features

### Lifetime Unlock
**Price:** $149.99 (one-time payment)
**Features:**
- ✓ All Premium Subscription features
- ✓ No recurring payments
- ✓ Lifetime updates
- ✓ Premium technical support

## Subscription Status

Users can check their subscription status in the **Settings** screen, which displays:
- Current subscription tier
- Active features
- Expiration date (for trial/monthly)
- Renewal information
- Upgrade options

## In-App Purchase Integration

The app uses platform-specific in-app purchase systems:
- **iOS:** StoreKit for App Store subscriptions
- **Android:** Google Play Billing for Play Store subscriptions

## Trial Period

- Trial starts automatically on first launch
- No credit card required for trial
- Full access to all features during trial
- Graceful downgrade to free features after trial expires
- Clear reminders before trial expiration

## Subscription Management

Users can manage subscriptions through:
1. **In-App Settings:** View status, upgrade, or change plans
2. **App Store/Play Store:** Cancel, modify, or restore subscriptions
3. **Support:** Contact support for subscription issues

## Feature Restrictions

When subscription expires or is not active:
- Advanced color modes show "Premium Required" overlay
- External stream input shows upgrade prompt
- Recording time limited to 5 minutes
- Gallery limited to last 5 recordings
- Motion detection reverts to basic sensitivity

## Restore Purchases

Users who previously purchased can restore their subscription:
1. Navigate to Settings → Subscription
2. Tap "Restore Purchases"
3. Sign in with App Store/Play Store account
4. Subscription features re-enabled automatically
